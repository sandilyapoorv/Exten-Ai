# Manual Test: Exten AI Chrome Extension

## Load Unpacked Extension

1. Open Chrome.
2. Go to `chrome://extensions`.
3. Enable Developer mode.
4. Click "Load unpacked".
5. Select this folder: `/home/primal/Sandilyapoorv/Exten-ai/chrome`.
6. Pin "Exten AI" from the extensions menu.

## Notes

- This folder is self-contained for the Chrome build.
- The QR image used by the support panel is included as `upi_qr.png`.
- Local test/check scripts use the files inside this folder only.
